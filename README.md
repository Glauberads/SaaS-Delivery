Uma Landing Page premium focada em conversão para o projeto **VIP Delivery**, uma plataforma SaaS (Software as a Service) no modelo White Label. O objetivo da página é captar parceiros, infoprodutores e agências que desejam ter seu próprio sistema de delivery para revender a negócios locais.

---

## 💻 Sobre o Projeto

A interface foi projetada utilizando conceitos avançados de UI/UX, com forte apelo "SaaS Premium" (Dark Mode, layout limpo, cores neon sutis e micro-interações). A meta é transmitir alto valor agregado, autoridade e profissionalismo.

### Principais Seções da Página
1. **Hero:** Promessa principal e CTAs focados em conversão.
2. **Dores do Público:** Identificação de problemas do modelo tradicional de prestação de serviços.
3. **Oportunidade:** Visão do modelo de negócio recorrente.
4. **Recursos (Features):** O que o cliente final recebe (Aplicativo, Dashboard, PDV).
5. **Projeção Financeira:** Calculadora estática de MRR baseada em número de clientes.
6. **Prova Social:** Avaliações de parceiros atuais.
7. **Pricing:** Tabela de preços clara com o plano PRO em destaque.

## 🚀 Como executar o projeto localmente

### Pré-requisitos
Certifique-se de ter o **Node.js** instalado na sua máquina (versão 18+ recomendada).

### Passo a Passo

1. **Clone o repositório**
   ```bash
   git clone https://github.com/Glauberads/SaaS-Delivery.git
   ```

2. **Acesse a pasta do projeto**
   ```bash
   cd SaaS-Delivery
   ```

3. **Instale as dependências**
   ```bash
   npm install
   ```

4. **Inicie o servidor de desenvolvimento**
   ```bash
   npm run dev
   ```

5. O projeto estará disponível em `http://localhost:5173`.

---

## 📦 Scripts Disponíveis no `package.json`

- `npm run dev`: Inicia o servidor local de desenvolvimento.
- `npm run build`: Compila a aplicação para produção (arquivos otimizados vão para a pasta `/dist`).
- `npm run lint`: Roda a checagem de erros utilizando o ESLint.
- `npm run preview`: Simula a execução do build final gerado localmente.
